# Getting Started Guide

## Installation

### Prerequisites
1. **Arduino IDE**: Download from [arduino.cc](https://www.arduino.cc/en/software)
2. **Arduino Board**: Uno, Nano, or Mega recommended
3. **USB Cable**: Type A to Type B for Arduino Uno

### Setting Up
1. Install Arduino IDE
2. Connect Arduino to computer
3. Select correct board and port in Tools menu

## First Project: LED Blink

### Components Needed
- Arduino Board
- LED (any color)
- 220Ω Resistor
- Breadboard
- Jumper wires

## Project Progression

Start with these projects in order:
1. LED Blink (Basic)
2. Traffic Light (Multiple LEDs)
3. Servo Control (Motor)
4. Ultrasonic (Sensor)
5. Temperature (LCD Display)
6. PIR Motion (Advanced)

## Troubleshooting

Common issues and solutions:
- **Port not found**: Reconnect USB or install drivers
- **Upload failed**: Check board selection
- **LED not blinking**: Check wiring polarity
