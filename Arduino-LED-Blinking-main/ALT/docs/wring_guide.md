# Wiring Guide – Arduino LED Blink


## Step-by-step wiring:
1. Place LED on breadboard
2. Connect **long leg (+)** to **220Ω resistor**
3. Connect resistor to **Arduino Pin 13**
4. Connect **short leg (-)** of LED to **GND**


---


## Diagram Logic
LED + (long leg) → Resistor → Pin 13
LED - (short leg) → GND
